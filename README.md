# 💬 Emitly — Real-Time Chat Application

A full-stack real-time chat application built with the MERN stack, featuring secure authentication, instant messaging, and a modern SaaS-inspired UI.

---

## 🚀 Project Overview

Emitly is designed as a production-ready messaging platform that supports real-time communication, presence tracking, and interactive user experience — similar to modern chat applications like WhatsApp Web and Discord.

## ✨ Core Features

* 🔐 **Custom JWT Authentication** (HTTP-only cookies, no third-party auth)
* ⚡ **Real-time Messaging** using Socket.io
* 🟢 **Online / Offline Presence Tracking**
* 💬 **Typing Indicators** with live updates
* ✔️ **Message Status** (Sent • Delivered • Seen)
* 🖼️ **Image Sharing** via Cloudinary
* 🔔 **Notification & Typing Sounds** (toggle enabled)
* 📨 **Automated Welcome Emails** using Resend
* 🚦 **API Rate Limiting** with Arcjet for security
* 🎨 **Modern UI** with React, Tailwind CSS & DaisyUI
* 🧠 **Global State Management** using Zustand

---

## 🧠 Engineering Highlights

* Designed a **real-time communication system** using WebSockets for low-latency messaging
* Implemented **secure authentication flow** with JWT and cookie-based sessions
* Built **optimistic UI updates** for instant message rendering
* Managed complex state using Zustand for scalable frontend architecture
* Structured backend with **modular REST APIs** and middleware-based security
